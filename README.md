# 葡萄病害识别微信小程序

## 项目介绍

这是一个基于微信小程序的葡萄病害识别系统，用户可以通过拍照或上传图片的方式识别葡萄的各种病害情况，系统会根据识别结果给出相应的建议、防治措施和用药注意事项，并提供购买链接。

## 功能特性

- 📷 **拍照识别**：通过摄像头拍摄葡萄病变部位进行识别
- 📱 **相册选择**：支持从相册选择已有的图片进行识别
- 🎯 **智能识别**：利用AI技术准确识别葡萄常见病害
- 💡 **专业建议**：根据识别结果提供详细的防治建议
- 💊 **用药指导**：推荐适合的农药及使用方法
- 🔗 **购买链接**：提供农药购买渠道
- 📤 **结果分享**：支持将识别结果分享给他人

## 技术栈

- **前端**：微信小程序原生开发（WXML, WXSS, JavaScript）
- **后端**：可对接任意AI图像识别API（如百度AI、腾讯云AI等）
- **数据存储**：微信小程序本地存储

## 开发环境搭建

1. **注册微信小程序账号**
   - 访问 [微信公众平台](https://mp.weixin.qq.com/) 注册小程序账号
   - 获取 AppID

2. **安装微信开发者工具**
   - 从 [微信开发者工具官网](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html) 下载并安装

3. **创建项目**
   - 打开微信开发者工具
   - 点击"新建项目"
   - 填写项目信息，使用本项目的代码

4. **配置 API 接口**
   - 修改 `app.js` 中的 `apiUrl` 和 `apiKey` 为实际的 API 地址和密钥

## 核心功能实现

### 1. 拍照上传功能

- 使用微信小程序的 `camera` 组件实现摄像头预览
- 调用 `wx.createCameraContext().takePhoto()` 方法拍照
- 使用 `wx.chooseImage()` 方法从相册选择图片
- 调用 `wx.uploadFile()` 方法上传图片到服务器

### 2. 病害识别功能

- 对接第三方图像识别 API
- 发送图片数据到 API 进行识别
- 解析 API 返回的识别结果

### 3. 结果展示功能

- 展示识别出的病害名称和置信度
- 显示病害症状描述
- 提供防治建议和用药指导
- 展示购买链接

### 4. 分享功能

- 使用 `wx.showShareMenu()` 方法显示分享菜单
- 实现 `onShareAppMessage()` 方法处理分享
- 实现 `onShareTimeline()` 方法分享到朋友圈

## 部署流程

1. **代码审核**
   - 在微信开发者工具中提交代码审核
   - 填写审核信息，说明小程序的功能和用途

2. **发布上线**
   - 审核通过后，在微信公众平台发布小程序
   - 配置小程序的基本信息，如名称、头像、简介等

3. **版本管理**
   - 后续更新可通过微信开发者工具提交新版本
   - 遵循微信小程序的版本管理规范

## 注意事项

1. **API 调用限制**
   - 部分第三方图像识别 API 可能有调用次数限制
   - 建议根据实际使用情况选择合适的 API 服务

2. **网络请求**
   - 小程序的网络请求需要在微信公众平台配置合法域名
   - 开发环境可开启 "不校验合法域名" 选项

3. **权限申请**
   - 使用摄像头需要申请 `scope.camera` 权限
   - 使用相册需要申请 `scope.album` 权限

4. **用户体验**
   - 确保识别过程中的加载动画流畅
   - 提供清晰的错误提示和操作指引
